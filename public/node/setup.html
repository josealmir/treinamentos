Setup do Projeto

Configuração inicial com TypeScript e Fastify

1

Inicialização do Projeto

# Criar diretório do projeto
mkdir fastify-api
cd fastify-api

# Inicializar projeto
npm init -y

Use npm init -y para aceitar defaults

2

Instalação de Pacotes

# Dependências de produção
npm i fastify @fastify/jwt @fastify/cors @fastify/sensible mysql2

# Dependências de desenvolvimento
npm i -D typescript ts-node-dev @types/node
3

Configuração TypeScript

{
  "compilerOptions": {
    "target": "ES2020",
    "module": "CommonJS",
    "outDir": "dist",
    "rootDir": "src",
    "strict": true,
    "esModuleInterop": true
  }
}
4

Scripts do Package.json

"scripts": {
  "dev": "ts-node-dev --respawn --transpile-only src/server.ts",
  "build": "tsc",
  "start": "node dist/server.js"
}
