Setup do Projeto

Configuração inicial com TypeScript e Fastify

TypeScript
Fastify

1️⃣ Inicialização do Projeto

Essencial
1# Criar diretório do projeto
2mkdir fastify-api
3cd fastify-api
4
5# Inicializar projeto
6npm init -y
Use npm init -y para aceitar defaults

2️⃣ Instalação de Pacotes

Dependências
1# Dependências de produção
2npm i fastify @fastify/jwt @fastify/cors @fastify/sensible mysql2
3
4# Dependências de desenvolvimento
5npm i -D typescript ts-node-dev @types/node

3️⃣ Configuração TypeScript

tsconfig.json
1{
2 "compilerOptions": {
3 "target": "ES2020",
4 "module": "CommonJS",
5 "outDir": "dist",
6 "rootDir": "src",
7 "strict": true,
8 "esModuleInterop": true
9 }
10}

4️⃣ Scripts do Package.json

Scripts
1"scripts": {
2 "dev": "ts-node-dev --respawn --transpile-only src/server.ts",
3 "build": "tsc",
4 "start": "node dist/server.js"
5}
