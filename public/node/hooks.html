Hooks do Fastify

Ciclo de vida das requisições e manipulação de eventos

Ciclo de Vida

onRequest Request

Executado no início da requisição

preValidation Validation

Antes da validação dos dados

preHandler Handler

Antes do handler principal

Exemplo de Hooks 

// src/hooks/auth.ts
app.addHook('onRequest', async (req, reply) => {
  // Autenticação inicial
  req.headers['x-start'] = Date.now().toString();
});

app.addHook('preHandler', async (req, reply) => {
  // Autorização e validação
  if (!req.user) {
    throw reply.unauthorized();
  }
});

app.addHook('onError', async (req, reply, error) => {
  req.log.error(error);
});

Hooks disponíveis:
onRequest, preValidation, preHandler, preSerialization, onSend, onResponse, onError
