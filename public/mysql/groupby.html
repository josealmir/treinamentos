DQL: GROUP BY e Agregações

Exemplos de como agrupar dados e aplicar funções de agregação com GROUP BY e HAVING:

-- Exemplo básico de agrupamento por categoria com contagem
SELECT category_id, COUNT(*) AS produtos_por_categoria
FROM products
GROUP BY category_id;

-- Agrupamento com múltiplas funções de agregação
SELECT
  category_id,
  COUNT(*) AS qtd_produtos,
  AVG(unit_price) AS preco_medio,
  SUM(units_in_stock) AS estoque_total,
  MIN(unit_price) AS menor_preco,
  MAX(unit_price) AS maior_preco
FROM products
GROUP BY category_id
HAVING COUNT(*) > 5
ORDER BY preco_medio DESC;

-- Agrupamento por múltiplas colunas
SELECT category_id, supplier_id, COUNT(*) AS produtos
FROM products
GROUP BY category_id, supplier_id
HAVING produtos > 2;

Pontos importantes:

  • GROUP BY: agrupa linhas com valores idênticos nas colunas especificadas
  • Funções de agregação: COUNT(), SUM(), AVG(), MIN(), MAX() - calculam valores sobre grupos
  • HAVING: filtra grupos após agregação (WHERE filtra linhas antes do GROUP BY)
  • Ordem de execução: FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY
