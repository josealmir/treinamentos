JSX - JavaScript XML

JSX é uma extensão de sintaxe para JavaScript que parece com HTML, permitindo escrever elementos da interface diretamente no código JavaScript.

O React utiliza JSX para descrever a aparência da UI. Na compilação, o JSX é transformado em chamadas React.createElement().

Exemplo:

// Declaração de componente usando JSX
    function Greeting({ name }) {
    const [isActive, setIsActive] = useState(false);
    
    return (
        <div className="greeting">
        <h1 style={{ color: isActive ? 'blue' : 'black' }}>
            Olá, {name}!
        </h1>
        <button onClick={() => setIsActive(!isActive)}>
            {isActive ? 'Desativar' : 'Ativar'}
        </button>
        </div>
    );
    }

    // Uso do componente
    <Greeting name="Maria" />

JSX facilita a criação de interfaces por combinar a lógica de renderização com a marcação da UI em um só lugar.
