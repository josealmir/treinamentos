Padrões Avançados em React

Higher Order Components

Funções que recebem um componente e retornam outro componente com funcionalidades adicionais.

Render Props

Compartilhamento de código através de uma prop que é uma função que retorna elementos React.

Compound Components

Conjunto de componentes que trabalham juntos, compartilhando estado implicitamente.

Exemplos:

// Exemplo de Higher Order Component

  return function WithLogger(props) {
    useEffect(() => {
      console.log('Componente montado');
      return () => console.log('Componente desmontado');

}
