Componentes em React

Componentes são os blocos de construção fundamentais no React. Permitem dividir a UI em peças independentes e reutilizáveis.

Existem dois tipos principais: Componentes Funcionais (mais modernos e recomendados) e Componentes de Classe (forma tradicional).

Componente Funcional

import React, { useState } from 'react';

    // Componente funcional com hooks
    function Contador() {
    const [contador, setContador] = useState(0);
    
    return (
        <div>
        <h2>Contagem: {contador}</h2>
        <button onClick={() => setContador(contador + 1)}>
            Incrementar
        </button>
        </div>
    );
    }

Componente de Classe

import React, { Component } from 'react';

    // Componente baseado em classe
    class Contador extends Component {
    constructor(props) {
        super(props);
        this.state = { contador: 0 };
    }
    
    incrementar = () => {
        this.setState({ 
        contador: this.state.contador + 1 
        });
    }
    
    render() {
        return (
        <div>
            <h2>Contagem: {this.state.contador}</h2>
            <button onClick={this.incrementar}>
            Incrementar
            </button>
        </div>
        );
    }
    }

Atualmente, componentes funcionais com Hooks são preferidos por serem mais simples, legíveis e facilitarem a reutilização de lógica.
