React Hooks: useState e useEffect

Os Hooks são funções que permitem utilizar recursos de React (como estado e ciclo de vida) em componentes funcionais.

useState: Gerencia estado local no componente
useEffect: Executa efeitos colaterais após renderização

Exemplos:

import React, { useState, useEffect } from 'react';

    function Contador() {
    // useState: [variável de estado, função para atualizar estado]
    const [contador, setContador] = useState(0);
    const [ativo, setAtivo] = useState(true);
    
    // useEffect executa após cada renderização
    useEffect(() => {
        // Este código roda após cada atualização do componente
        document.title = `Você clicou ${contador} vezes`;
        
        // Função de limpeza (opcional) - roda antes da próxima execução
        return () => {
        document.title = 'React App';
        };
    }, [contador]); // Array de dependências - executa apenas quando contador muda
    
    return (
        <div>
        <p>Você clicou {contador} vezes</p>
        <button onClick={() => setContador(contador + 1)}>
            Aumentar contagem
        </button>
        <button onClick={() => setAtivo(!ativo)}>
            {ativo ? 'Desativar' : 'Ativar'}
        </button>
        </div>
    );
    }

Hooks devem ser chamados apenas no nível superior do componente, nunca dentro de loops, condições ou funções aninhadas.
