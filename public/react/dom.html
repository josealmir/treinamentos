Virtual DOM

O Virtual DOM é uma representação leve em memória do DOM real. React usa esta abstração para otimizar o processo de renderização.

Como funciona:

  • React cria uma cópia virtual da árvore DOM
  • Quando o estado muda, cria uma nova árvore virtual
  • Compara as árvores usando algoritmo de reconciliação
  • Atualiza apenas os nós alterados no DOM real

Resultado: renderização mais eficiente e melhor performance.
