Ciclo de Vida dos Componentes

Todo componente React passa por diferentes fases durante sua vida útil. O entendimento deste ciclo é fundamental para o desenvolvimento eficiente.

Montagem (Mounting)

Componente é inserido no DOM

Atualização (Updating)

Componente é re-renderizado quando props ou state mudam

Desmontagem (Unmounting)

Componente é removido do DOM

No React moderno com Hooks, o useEffect substitui os métodos de ciclo de vida:

useEffect(() => { ... }, []); // componentDidMount
useEffect(() => { ... }); // componentDidUpdate
useEffect(() => { return () => { ... }; }); // componentWillUnmount
