Otimização de Performance em React

Conforme sua aplicação React cresce, a otimização de performance torna-se essencial. Técnicas fundamentais incluem memoização, carregamento sob demanda e divisão de código.

1. React.memo - Evita renderizações desnecessárias

// Memoização de componente para evitar renderizações
    const ExpensiveComponent = React.memo(({ data }) => {
    // Renderização apenas quando props mudam
    return (
        <div>
        {data.map(item => <ItemCard key={item.id} {...item} />)}
        </div>
    );
    });

2. Lazy Loading - Carregamento sob demanda

// Importação dinâmica de componentes
    import { Suspense, lazy } from 'react';

    // O componente só é carregado quando necessário
    const Dashboard = lazy(() => import('./Dashboard'));

    function App() {
    return (
        <Suspense fallback={<div>Carregando...</div>}>
        <Dashboard />
        </Suspense>
    );
    }

3. Code Splitting - Divisão de pacotes

// Usando code splitting com React Router
    import { BrowserRouter, Routes, Route } from 'react-router-dom';
    const Home = lazy(() => import('./Home'));
    const About = lazy(() => import('./About'));
    const Shop = lazy(() => import('./Shop'));

    // Cada rota gera um pacote JavaScript separado
    <BrowserRouter>
    <Suspense fallback={<Loader />}>
        <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/shop" element={<Shop />} />
        </Routes>
    </Suspense>
    </BrowserRouter>

Estas técnicas reduzem o tempo de carregamento inicial e melhoram a experiência do usuário ao carregar apenas o necessário.
