Props e State

Props (propriedades) permitem a passagem de dados de um componente pai para um filho. São imutáveis e funcionam como parâmetros de função.

State armazena dados que podem mudar ao longo do tempo dentro de um componente. Quando o state é atualizado, o React re-renderiza o componente automaticamente.

Props são para comunicação entre componentes, enquanto State é para gerenciamento interno. A combinação de ambos forma a base da reatividade do React.
