Arrays vs Objetos e Estruturas Avançadas

JavaScript oferece diferentes estruturas para organizar dados, cada uma com propósitos e características específicos.

Arrays Objetos
Indexados por númerosIndexados por chaves/nomes
Ordem mantidaSem ordem garantida
Métodos: push, map, filterAcesso: dot notation, bracket
Para listas de itensPara dados estruturados

Estruturas Avançadas

  • Typed Arrays: Arrays com tipos específicos (Int8Array, Float32Array)
  • Map: Coleção de pares chave-valor com chaves de qualquer tipo
  • Set: Coleção de valores únicos
  • WeakMap e WeakSet: Versões com referências fracas

Exemplos de Código:

// Array - Lista ordenada de elementos
const alunos = ['Ana', 'Pedro', 'Carla'];
alunos.push('Miguel');
const aprovados = alunos.filter(a => a !== 'Pedro');

// Objeto - Coleção de propriedades
const pessoa = {
  nome: 'Ana',
  idade: 28,
  profissao: 'Desenvolvedora',
  saudar() {
    return `Olá, sou ${this.nome}`;
  }
};

// Estruturas avançadas
const numeros = new Int32Array([1, 2, 3]);
const usuarios = new Map();
usuarios.set('admin', { nivel: 'root' });

Escolha a estrutura adequada para cada problema: arrays para coleções sequenciais e objetos para entidades com atributos nomeados.
