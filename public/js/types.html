Tipos de Dados e Sintaxe Básica

JavaScript possui diferentes tipos de dados primitivos e objetos. Os primitivos são os blocos fundamentais para construção de estruturas mais complexas.

Tipos Primitivos

  • Number Números inteiros ou decimais
  • String Texto entre aspas simples, duplas ou crases
  • Boolean Valores lógicos true ou false
  • Null Ausência intencional de valor
  • Undefined Valor não atribuído

Características

  • JavaScript é fracamente tipado
  • Conversão automática entre tipos
  • Operador typeof para verificar tipos
  • Tipos complexos: Object, Array, Function

Exemplos de Código:

// Números (Number)
let inteiro = 10;
let decimal = 10.5;
let negativo = -42;

// Strings
let nome = 'Ana';
let sobrenome = "Silva";
let frase = `${nome} tem ${inteiro} anos`;

// Booleanos
let ativo = true;
let maiorIdade = inteiro >= 18;  // true

// Null e Undefined
let usuario = null;
let endereco;  // undefined

// Verificando tipos
console.log(typeof inteiro);     // "number"
console.log(typeof nome);        // "string"
console.log(typeof ativo);       // "boolean"

Em JavaScript, o operador typeof null retorna "object", o que é considerado um erro histórico da linguagem. Na verdade, null é um tipo primitivo.
