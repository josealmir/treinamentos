Objetos: Propriedades e Métodos

Objetos em JavaScript são coleções de pares chave-valor que armazenam propriedades e métodos relacionados.

Criação de Objetos

  • Notação literal: { }
  • Construtor: new Object()
  • Object.create()
  • Classes (ES6+)

Manipulação de Objetos

  • Acessar propriedades: objeto.propriedade ou objeto["propriedade"]
  • Adicionar/modificar propriedades
  • Deletar com delete objeto.propriedade
  • Métodos úteis: Object.keys(), Object.values(), Object.entries()

Exemplo de Código:

// Criando um objeto pessoa
const pessoa = {
  nome: "Ana Silva",
  idade: 28,
  profissao: "Desenvolvedora",
  
  // Método dentro do objeto
  apresentar: function() {
    return `Olá, meu nome é ${this.nome} e tenho ${this.idade} anos`;
  },
  
  // Método com sintaxe abreviada (ES6)
  programar() {
    return `${this.nome} está programando em JavaScript`;
  }
};

// Acessando propriedades e métodos
console.log(pessoa.nome);         // "Ana Silva"
console.log(pessoa.apresentar()); // "Olá, meu nome é..."

Os objetos são a base da programação orientada a objetos em JavaScript e permitem organizar dados e comportamentos relacionados em uma única estrutura.
