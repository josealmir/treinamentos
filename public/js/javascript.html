JavaScript

O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, interpretada, que permite adicionar interatividade às páginas web.

Características Principais

  • Linguagem de script do lado do cliente
  • Interpretada pelo navegador
  • Orientada a objetos e eventos
  • Sintaxe baseada em C, Java e outras

Usos Comuns

  • Manipulação do DOM
  • Validação de formulários
  • Animações e efeitos visuais
  • Consumo de APIs e dados externos

Exemplo de Código:

// Exemplo básico de JavaScript
let mensagem = "Olá, mundo!";
const container = document.getElementById('container');

// Função para exibir uma saudação
function saudacao(nome) {
    return `Bem-vindo, ${nome}!`;
}

// Adicionando evento ao botão
document.getElementById('botao').addEventListener('click', () => {
    container.innerHTML = saudacao("Estudante");
    console.log(mensagem);
});

JavaScript é executado diretamente no navegador e não precisa de compilação, tornando o desenvolvimento web mais ágil.
