Arrays: Manipulação e Métodos

Arrays são estruturas de dados que armazenam múltiplos valores em uma única variável. São fundamentais para manipular conjuntos de dados em JavaScript.

Métodos Principais

  • push/pop: Adiciona/remove elementos no final
  • unshift/shift: Adiciona/remove no início
  • map: Cria novo array transformando elementos
  • filter: Filtra elementos com base em condição
  • forEach: Executa função para cada elemento

Características

  • Indexados a partir do zero
  • Podem conter diferentes tipos de dados
  • Tamanho dinâmico
  • Métodos para transformação e iteração

Exemplos de Código:

// Criando e manipulando arrays
const frutas = ['Maçã', 'Banana', 'Laranja'];
frutas.push('Morango');  // Adiciona ao final

// Usando método map para transformar dados
const numeros = [1, 2, 3, 4, 5];
const dobrados = numeros.map(numero => numero * 2);
// dobrados: [2, 4, 6, 8, 10]

// Usando método filter para filtrar elementos
const pares = numeros.filter(numero => numero % 2 === 0);
// pares: [2, 4]

// Combinando métodos em cadeia
const resultado = numeros
  .filter(n => n > 2)
  .map(n => n * 3);
// resultado: [9, 12, 15]

Os métodos de array em JavaScript são poderosos para manipulação de dados e programação funcional. Dominar esses métodos é essencial para um código mais limpo e eficiente.
