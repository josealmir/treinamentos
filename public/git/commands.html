Comandos Essenciais do Git

Comandos Básicos

git init

Inicializa um novo repositório Git no diretório atual.

$ git init

git add

Adiciona arquivos à área de staging (preparo para commit).

$ git add arquivo.txt
$ git add .

git commit

Registra as alterações da área de staging no repositório.

$ git commit -m "Mensagem descritiva"

git status

Mostra o estado do repositório: arquivos modificados, adicionados, etc.

$ git status

Colaboração e Visualização

git clone

Clona um repositório remoto para o seu computador.

$ git clone https://github.com/usuario/repo.git

git pull

Obtém atualizações do repositório remoto e as integra ao local.

$ git pull origin main

git push

Envia suas alterações locais para o repositório remoto.

$ git push origin main

git log

Exibe o histórico de commits do repositório.

$ git log
$ git log --oneline

Dica importante

Use o comando git --help para obter ajuda sobre qualquer comando Git ou git comando --help para ajuda específica sobre um comando.
