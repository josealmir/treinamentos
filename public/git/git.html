Git

O que é Git?

Definição

O Git é um sistema de controle de versão distribuído, gratuito e de código aberto, projetado para lidar com projetos de qualquer tamanho com velocidade e eficiência.

História

Criado por Linus Torvalds em 2005 para o desenvolvimento do kernel Linux, após disputas com o sistema BitKeeper.

Por que é tão popular?

  • Gratuito e open-source
  • Rápido e eficiente mesmo com grandes projetos
  • Funciona offline (sem necessidade de conexão contínua)
  • Adotado por empresas e projetos de todos os tamanhos

Características Principais

Distribuído

Cada desenvolvedor tem uma cópia completa do repositório, incluindo histórico

Integridade

Todo conteúdo é verificado com hash SHA-1, protegendo contra corrupção

Staging Area

Área intermediária para preparar mudanças antes de confirmar (commit)

Branching

Criação de ramificações leves e rápidas para trabalhos paralelos

Diferencial

Diferente de outros VCS como SVN e CVS, o Git armazena "snapshots" de arquivos completos em vez de apenas diferenças, tornando operações mais rápidas.
