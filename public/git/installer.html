Git

Instalação e Configuração do Git

Instalação por Sistema

Windows

  1. Acesse git-scm.com
  2. Baixe o instalador para Windows
  3. Execute e siga as instruções do assistente
  4. Selecione o editor padrão durante a instalação

macOS

  1. Via Homebrew: brew install git
  2. Ou baixe o instalador do site oficial
  3. Ou instale Xcode Command Line Tools: xcode-select --install

Linux

Debian/Ubuntu: sudo apt-get install git

Fedora: sudo dnf install git

Verificar Instalação

git --version

Configuração Inicial

Após instalar o Git, configure seu nome de usuário e email:

Configurar identidade

git config --global user.name "Seu Nome"

git config --global user.email "seu@email.com"

Configurações Adicionais

Ativar cores

git config --global color.ui true

Editor padrão

git config --global core.editor "nome-do-editor"

Listar configurações

git config --list

Configuração por Projeto

Para configurar apenas um repositório específico, remova a flag --global e execute o comando dentro da pasta do projeto.
